# Source Word Lists

Raw CEFR-tagged word lists that seed the pipeline (`pipeline/clean.ts` input).
Columns: `headword,pos,CEFR,CoreInventory 1,CoreInventory 2,Threshold`.

Not committed (see `.gitignore`) — these are large, locally curated CSVs. Ask
the project owner for the current `ALL.csv` / `A1.noun.csv` files, or substitute
your own word list with the same columns.
