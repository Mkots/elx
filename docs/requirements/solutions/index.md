# Solutions

Top-level product definitions that requirements refine.

## Solutions

| ID                            | Name                    | Description                                                         |
| ----------------------------- | ----------------------- | ------------------------------------------------------------------- |
| [SOL-LEXTALE](SOL-LEXTALE.md) | LexTALE vocabulary test | Web application assessing English vocabulary via the LexTALE method |

## Scope

SOL-LEXTALE covers:

- **Core flow:** word selection (Stage 1) → verification and scoring (Stage 2)
- **Optional challenges:** synonyms/antonyms, spelling, meaning, semantic usage
- **Platform:** SSR/MPA architecture, session state, PostgreSQL question bank
- **Operations:** deployment, backups, observability, quality gates
